TaskJar Software for task management

  • Live Demo
  • Source code

A project management software that helps with management of tasks at work. Users can add projects, add project tasks and then they can track the task progress as they work, based on task urgency.

I used django for the backed and APIs and then consumed the apis with react, which I use to build the frontend of the application. I also used redux for state management.

  • Learnt and managed to integrate my React and Django applications effectively
  • Managed to integrate multiple features including an authentication system using Json Web Tokens
  • Managed to deploy the application to the web
  • Hosted my postgres database on the complex AWS-RDS service
