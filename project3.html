Estudiar Web App | Online Learning Platform

  • Live Demo
  • Source code

A web application where users can create chat-rooms/study-rooms to help each other with learning their specific areas of interest.

I used django for the backed and APIs and then consumed the apis with react, which I use to build the frontend of the application. I also used redux for state management. Additionally I used bootswatch's Lux bootstrap theme for the frontend components

  • Learnt best practices for building fullstack applications
  • Understood how to manage many-to-many relationships in relational databases
